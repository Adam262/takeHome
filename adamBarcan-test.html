  • Q: Describe the differences between the IE6, IE7, and W3C box models.

    CSS Box Model

    The CSS box model represents each html element on a page as a box with four possible properties: content, border, padding and margin. The below browser models differ on how these properties contribute to an element's height or width.

    IE6 box model

    In the IE6 box model, an element's default height or width property reflects the total combined height or width of its content, left and right borders and left and right padding. You can think of IE6 width as an element's visible height or width. For example, a div with width: 300px will continue to have a 300px width, even if a border or padding are added. Only the addition of margin would add to the element's width.

    IE7 box model

    The IE7 box model adopts the approach of the W3C box model, as described below.

    W3C box model

    In the W3C box model, an element's default width property reflects the width of its content only. Border, padding or margin are then added to content width. One can rever to IE6 box-model by styling an element with "box-sizing: border-box."
